0.5.0

Many improvements for 0.5.0:

- Search notes using sqlite's full text search capabilities (AND, OR NOT, NEAR(), etc...)
- The time bar looks better and includes the most recent month and a way to jump to the most recent notes
- Clicking on a related note now loads that note correctly instead of just scrolling up
- Implement back navigation after clicking on a related note
- Add note if no note is selected, and add note with a relation
- Show fewer threads, and other improvements to threads
- Long links do not break layout

0.4.0

This is a significant rejigger of the Illuminoted app!

- Threads are in sidebar on left side of app (can be pinned)
- Threads are loaded incrementally and are ordered by last activity
- App can be refreshed and preserve selected threads (using URL parameters)
- Use browser history to go back to previous selected threads
- Threads can be edited and moved
- New Note edit interface (including changing thread)
- Fixed bug in relations editing
- Note listing preserves focus date on change of threads or search
- "Show notes in all threads" feature
